Mundo dos Tubarões

Bem-vindo ao mundo dos tubarões das profundezas. Este site foi feito para catalogar as criaturas mais fascinantes do oceano.

Espécies Catalogadas

  • Tubarão-Duende (Mitsukurina owstoni)
  • Tubarão-Fantasma (Chimaera monstrosa)
  • Tubarão-da-Groenlândia (Somniosus microcephalus)

Tubarão

O tubarão é um peixe da classe dos Chondrichthyes, habita os mais diversos ambientes oceânicos e ocupa os níveis superiores das cadeias alimentares dos ambientes marinhos.
